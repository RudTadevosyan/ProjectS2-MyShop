# MyShop

## Project Overview
**MyShop** is an educational e-commerce web application based on the book 
*"Django 2 by Example"* by Antonio Mele"  
The project simulates a small online shop where users can browse products, add items to their cart, and complete a checkout process.  
This repo was used as part of a university assignment to study Python and gain practical experience building web applications.

## Features
- **Product Catalog:** Organized by categories with product details and pricing.  
- **Shopping Cart:** Add, remove, and update items before checkout.  
- **Admin Panel:** Customized admin interface to manage products, categories, and orders.  
- **Order Management:** Users can place orders and view order history.  
- **Frontend:** HTML and CSS templates with basic JavaScript for interactivity.  

## Technologies Used
- **Backend:** Python 3.x, Django 2.x  
- **Database:** SQLite (for local development)  
- **Frontend:** HTML, CSS, JavaScript  
- **Tools:** Git and GitHub for version control  

## Learning Outcomes
Working on this project helped me understand and practice:  
- Django project structure and modular apps  
- Defining models, views, and templates, and connecting them with URLs  
- Implementing authentication, sessions, and user permissions  
- Handling static files and integrating basic frontend interactivity  
- Building core e-commerce functionality like shopping carts and checkout  
- Working with forms, CRUD operations, and database migrations  
- Deploying and running a Django project locally
